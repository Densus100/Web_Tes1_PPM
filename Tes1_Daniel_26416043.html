Rangkuman Materi Git Hub

1. Apabila kita akan membuat Git repository dalam sebuah folder maka ketik "git init" setelah kita ada di dalam folder tersebut

2. Apabila ingin melihat status di Git repository maka ketik "git status". kita dapat melihat mana file yang masih untracked dan mana file yang sudah di staging area

3. Untuk melakukan "save" terhadap perubahan dan memberikan comment pada setiap langkah yang kita lakukan maka ketik "git commit -m "comment""

4. Untuk menambahkan sebuah file dari untracked ke staging area maka ketik "git add (nama file)", sedangkan untuk menambahkan banyak file yang memiliki format sama maka ketik "git add 'format'"

5. Untuk menghapus sebuah file maka ketik "git rm (nama file)", sedangkan untuk menghapus banyak file yang memiliki format sama maka ketik "git rm 'format'"

6. Untuk melihat history / log mengenai apa saja yang telah kita lakukan maka ketik "git log"

7. Untuk menambahkan alamat dari GitHub repository kedalam local git repository agar dapat di gunakan oleh banyak user maka ketik "git remote add (remote name) https:(web.git)"

8. Jika kita ingin menambahkan perubahan dari local Git repository kita ke dalam GitHub maka ketik "git push -u (remote name) (local branch name)" default dari local branch name adalah master (-u berfungsi agar tidak perlu mengetik ulang remote name dan local branch name, cukup mengetik "git push")

9. Jika ingin menambahkan perubahan dari GitHub ke local Git repository maka ketik "git pull (remote name) (local branch name)"

10. Jika ingin melihat perbedaan dari commit yg sebelumnya maka ketik "git diff HEAD'

11. Jika ingin mengetahui perbedaan file yg masuk ke stagging area di bandingkan commit yang sebelumnya maka ketik "git diff --staged"

12. Apabila ingin menghapus sebuah file dari stagging area maka ketik "git reset (nama file)"

13. Apabila ingin mengembalikan sebuah file ke commit yg sebelumnya maka ketik "git checkout -- (target file)"

14. Apabila ingin membuat branch copy dari branch yang sedang di kerjakan / aktif maka ketik "git branch (nama branch baru)"

15. Untuk menuju ke branch lain agar dapat melakukan perubahan maka ketik "git checkout (nama branch lain)"

16. Untuk melakukan copy dari branch yang lain ke branch yang aktif sekarang maka ketik "git merge (nama branch lain)"

17. Untuk menghapus branch yang lain maka ketik "git branch -d (nama branch lain)"
